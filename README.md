# EE106A Final Project Website: ROAR

### Environment Setup 

Virtual Environment
```
$ python3 -m venv 106a_env
$ source 106a_env/bin/activate
```

Conda Environment
```
$ conda create -n 106a_env python=3.6.9
$ conda activate 106a_env
```

### Install

```
$ pip install -r requirements.txt
```

### Usage

```
$ streamlit run main.py
```
